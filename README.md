# auto-cleanrl

Autonomous research on Atari with [CleanRL](https://github.com/vwxyzjn/cleanrl), driven by [xgenius](https://github.com/roger-creus/xgenius).

An AI agent (Claude Code) autonomously formulated 45+ hypotheses, wrote code, ran experiments on SLURM clusters, and iterated — with no human intervention.

## Key files

- [`research_goal.md`](research_goal.md) — the research objective given to the agent
- [`.xgenius/journal.md`](.xgenius/journal.md) — the agent's full research journal
- [`report/report.md`](report/report.md) — auto-generated research report
- [`results/`](results/) — raw experiment results (CSV)
